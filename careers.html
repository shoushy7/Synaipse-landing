Founding Lead Engineer (Employee #1)

Remote / Hybrid (Cincinnati, OH) Full-Time, Founding Team Competitive Salary + Founding Equity (1% – 5%)

As our Founding Lead Engineer, you will own the technical roadmap. You aren't just building features — you are architecting the foundation of a healthcare transformation engine. You will report directly to the CEO and President.

What You Will Do

  • Architect & Build: Lead the development of our MVP, focusing on RAG-based LLM pipelines and HIPAA-compliant cloud architecture.
  • Product Evolution: Work alongside neurosurgeons to translate complex clinical workflows into seamless automation tools.
  • Security First: Implement and maintain rigorous data security standards (HIPAA/SOC2) from day one.
  • Scale: Set the engineering culture. Recruit, mentor, and lead the future engineering team as we grow.

What We're Looking For

  • "0-to-1" Experience: You have built and shipped products from scratch. You thrive in ambiguity.
  • Technical Stack: Proficiency in Python (FastAPI/Flask), TypeScript/Node, React, and experience with Vector Databases (Pinecone/Weaviate) and LLM Orchestration (LangChain/LlamaIndex).
  • Healthcare Context: Deep understanding of PHI, HIPAA compliance, and ideally, experience with EHR integrations (HL7/FHIR).
  • Execution over Ego: You write clean code fast. You prioritize solving the problem over using the "coolest" new framework.

Founding Senior Frontend Engineer

Cincinnati, OH (Hybrid Preferred) Full-Time, Founding Team Significant Equity Component

Your mission is to build the "Command Center" — a high-velocity React interface that consolidates Fax, Email, and Portal data into a single, lightning-fast workflow for surgical intake teams.

Key Responsibilities

  • Architect the "Command Center": Design and build a complex, data-heavy dashboard using React, TypeScript, and Tailwind. It must be fast, keyboard-navigable, and dense with information.
  • The "Click-to-Verify" Engine: Build the PDF interaction system — rendering PDFs in the browser and creating a coordinate-mapping layer that allows AI to highlight specific text.
  • Invisible UX: Design workflows that require zero training. If a user has to read a manual, we failed.
  • EHR Integration (Frontend): Build "Headless" UI components that visualize data scraped from legacy systems.

Who You Are

  • The "0 to 1" Builder: You have built complex SPAs from scratch. You don't need a Jira ticket to know what to do next.
  • Visual Thinker: You understand that in high-volume operations, density and speed matter more than white space.
  • Technical Stack — Expert: React, TypeScript, CSS/Tailwind. Required: Canvas API, PDF rendering (react-pdf / pdf.js), or complex DOM manipulation. Bonus: Electron or Chrome Extensions.
  • Egoless: Ready to work directly with two neurosurgeon-founders who value logic, speed, and clinical accuracy.
